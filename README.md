# AI 赋能开发 | 相亲市场竞争力测评网站

一款多维度婚恋价值测评互动网站，依托大数据评估逻辑，整合用户学历、收入、外形、家庭资产、地域背景等核心维度，实现个性化相亲竞争力评分与可视化结果反馈。

## 项目亮点

- **多维度测评** - 整合学历、收入、外形、家庭资产、地域背景等核心维度
- **智能评分系统** - 基于大数据评估逻辑，精准生成婚恋市场定位报告
- **可视化结果** - 评测结果以图表形式直观展示
- **省市二级联动** - 省份-城市选择器，数据精准
- **响应式设计** - 完美适配手机端和电脑端
- **AI 辅助开发** - 全程借助 AI 工具完成模块化开发与功能调试

## 技术栈

- **前端**: HTML5 + CSS3 + JavaScript
- **AI 工具**: MiMoCode 智能体协作开发
- **部署**: GitHub Pages

## 功能特性

1. 用户信息录入（学历、收入、外形、家庭资产、地域等）
2. 智能竞争力评分计算
3. 专属婚恋市场定位报告生成
4. 可视化数据图表展示
5. 测评结果分享功能
